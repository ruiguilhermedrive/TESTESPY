🐛 TROUBLESHOOTING RÁPIDO
═════════════════════════════════════════════════════════════════════════════

Erro: "ModuleNotFoundError: No module named 'streamlit'"
→ pip install streamlit --break-system-packages

Erro: "Porta 8501 já está em uso"
→ streamlit run app_backtest.py --server.port 8502

Erro: "Dados não carregam"
→ Sistema usa dados simulados automaticamente (normal!)

Gráficos não aparecem
→ Atualize a página (F5) ou limpe cache:
  streamlit cache clear


⚡ USO PROGRAMÁTICO (Python)
═════════════════════════════════════════════════════════════════════════════

Para usar o sistema em seu próprio código:

    from spy_backtest_system import StrategyBacktester
    
    # Criar backtester
    bt = StrategyBacktester(ticker='SPY', days=90)
    
    # Executar backtest
    resultados = bt.backtest_parametros([-0.5, -1.0, -1.5])
    
    # Ver resultados
    print(resultados)
    
    # Gerar gráficos
    bt.plot_comparison_buy_hold(-0.5)
    
    # Exportar trades
    bt.export_trades(-0.5, 'meus_trades.xlsx')
